Tutorial

Selamat datang di Nyobain ML! Mulailah dengan melihat tutorial langkah demi langkah kami atau masuk ke tampilan utama.

1. Klik Unggah Gambar.

2. Atur besar atau kecilnya ukuran border dengan cara menggeser.

3. Pilih warna border selera Anda, ada 3 warna; Merah, Biru, Hijau.

4. Klik Unduh untuk menyimpan hasil twibbon.
